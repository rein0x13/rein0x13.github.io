The second one is Microsoft Windows 11 IoT Enterprise LTSC (if you know, you know...) for software not yet (or might never be) natively supported on Linux. The third one is generally where I test out different linux distributions (planning to learn BSD-based distros in my free time soon too) after installing them in a virtual machine, experiencing their bare-metal performance. I value trust not only in people but also in software. I have been using Microsoft Windows in my whole computing life since Windows XP down to Windows 11. I have previously joined the Windows Insider program to test out new developments and provide feedback before the Fall Creators Update was released back in 2017. Unfortunately, the more I used Windows, the more I was exposed to its shady natures. From forced online accounts, system-level advertisements, aggressive telemetry, automatic non-security breaking updates, it's certainly heart-breaking what Windows has become. The tipping point was when Microsoft pushed an update that corrupted my previous laptop's Windows 11 install beyond repair. Lots of missing and corrupted system files, ACPI and booting issues, and broken windows recovery, trust in Microsoft fading - that is when I started considering using a linux-based operating system instead whenever I can.

Fast forward 4 months after, I bought a second hand Lenovo Thinkpad X1 Carbon Gen 8 and has been my daily driver. I have tried multiple linux distributions and desktop environments each with their own ups and downs... and politics. From the flavors of Ubuntu Interim and LTS releases, Debian, Fedora, and CachyOS to the different desktop environments such as Gnome, KDE Plasma, XFCE, LXQT, Sway, and Niri WM. Despite choosing a thinkpad, my experience was not without major setbacks. However, I have loved the experience overall and have fallen in love with Free and Open Source Software.
